2020 Census Response Rate

Date Thu 11 June 2020 Category Viz Tags census / map

Recently census.gov has begun releasing the response rate for the 2020 census. This map shows the percentage of people who responded in each city/town, as well as an option to view how many of those that responded, did so by using the internet. Future plans to update the map allowing drill down to the census-tract and census-block-group detail, but currently the very granular geographies are missing some info.
