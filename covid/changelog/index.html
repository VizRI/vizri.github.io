Changelog

All notable changes to this project will be documented here.

2021-09-24 - CMS Table Fix + Vaccine Added

Added
  • Staff Vaccination data showing the most recent numbers of vaccinated staff by provider.
  • Staff Weekly Vaccine data showing a running total number of nursing home employees vaccinated.
  • Resident Vaccination data showing the most recent numbers of vaccinated residents by provider.
  • Resident Weekly Vaccine data showing a running total number of nursing home residents vaccinated.
Changes
  • Added CMS.gov nursing home and CDC.gov vaccination data details to the About page and provided links to all sources.
  • All Nursing Home tables were broken due to the change of the datasource location, new source located at data.cms.gov.
  • Additionally some fields were renamed breaking the code that generates the tables which have been fixed.

2021-03-08 - Vaccine Update

Added
  • Vaccine daily update data from CDC highlighting vaccination numbers.
  • Vaccination trend from RIDOH showing 1 and 2 doses administered and a 7-day moving average.
  • CDC Percent vaccine used showing how much recieved vaccine each state has used.
  • CDC 1 Dose showing how RI ranks compared to other states in people who have recieved 1 dose.
  • CDC Fully vaccinated showing how RI ranks compared to other states in people who have recieved 2 doses.
  • CDC 1 Dose (65 plus) showing how RI ranks compared to other states in people 65+ who have recieved 1 dose.
  • CDC Fully vaccinated (65 plus) showing how RI ranks compared to other states in people 65+ who have recieved 2 doses.
Changes
  • RIDOH vaccine numbers now appear in Daily Update showing new partially and fully vaccinated.
  • RIDOH vaccine numbers now appear in Current Totals showing total partially and fully vaccinated.
  • Added a new graph new deaths highlighting just today's deaths.
  • Added a new graph new people tested showing what % of tests are first time takers.

2021-02-19 - Nursing Homes Update

Added
  • Nursing Home weekly update data from CMS.gov highlighting cases & deaths in LTC facilities.
  • Tables for deaths normalized by both percent and rate (% and rate residents only).
  • Tables for cases of confirmed positive COVID-19 and rate among residents.
  • Table showing capacity of facilities showing how many beds are occupied.
  • Tables for both PPE supplies and staffing shortages.
  • Timeline tables of deaths, cases and capacity by week-end.
Changes
  • All tables should now start sorted by default highest to lowest of main metric.
  • Added date to the top right of all tables highlighting when the data is from.

2021-02-01 - Schools Update

Added
  • Schools weekly update page highlighting cases & number of schools with under 5 cases.
  • Tables for student and staff cases for in-person/hybrid schools.
  • Tables for student and staff cases for virtual schools.
Changes
  • Daily update sources to be more uniform with the naming of values.
  • Added column names from RIDOH sheet to source information modals.

2021-01-13 - Project Release

Added
  • Daily update page tracking important information & deaths.
  • Current totals page tracking totals for various metrics.
  • Deaths graphs showing daily deaths and change overtime.
  • Testing graphs showing positive cases since March 2020.
  • Hospitalized graphs showing admissions and icu/vent numbers.
